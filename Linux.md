## What is Linux?

Linux is an operating systems that's extremely popular, especially for servers in data centers, cloud computer, cybersecurity, internet of things.

- Linux is the kernel of a family of open-source Unix like operating systems (OS)
	- OS: A system software that manage computer hardware and provides various services for computer programs
	- Unix: A powerful, multitasking, multi-user OS that serves as the basis for many modern OS

## What is the GNU project?

- GNU stands for GNU's Not Unix:
	- The goal: To create a free and open-source operating system 
	- Initiated in 1983
	- Most GNU software is released under the GNU Feneral Public License (GPL), promoting the freedom to use, modify, and distribute the code
	- The Linux kernel, created in 1993 by Linus Torvalds, is released under GPL
	- Linux provides the kernel 
	- Together: GNU/Linux
