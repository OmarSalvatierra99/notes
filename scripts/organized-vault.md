# Vault Organizer — Minimal Guide

## Rules
- PDFs → `pdf/`
- Images → `media/`
- `#developer` → `developer/`
- `#art` → `art/`
- `#ideas` → `brain/`
- `#auditoria` → `auditoria-gubernamental/`

- Hidden folders and `.obsidian/` are skipped.
